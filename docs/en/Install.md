# Installation Guide

## 1. Download Extension

### Option A: Download from Releases (Recommended)

1. Visit the [Latest Release](https://github.com/MizuhaHimuraki/github-name-mapper/releases/latest) page.
2. Download the `github-name-mapper-vX.X.X.zip` file.
3. Extract it to a local folder.

### Option B: Clone Repository

```bash
git clone https://github.com/MizuhaHimuraki/github-name-mapper.git
```

## 2. Load into Chrome

1. Open Chrome browser and navigate to `chrome://extensions/`.
2. Toggle **Developer mode** in the top right corner.
3. Click **Load unpacked** button.
4. Select the `github-name-mapper` directory you just downloaded/extracted.

![Developer Mode](../img/install/chrome-extensions-1.png)

## 3. Verify Installation

After loading, you should see the **GitHub Name Mapper** extension card in your extensions list.

You can now click the extension icon in the toolbar to access the configuration.

![Load Unpacked](../img/install/chrome-extensions-2.png)
